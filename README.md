# ForgeMind AI

Enterprise Industrial AI Copilot powered by RAG, Google Gemini, FastAPI, React, and Qdrant.

## Features

- PDF Upload
- Semantic Search
- Evidence-backed Responses
- Root Cause Analysis
- Maintenance Advisor
- Agentic AI Workflow
- Docker Deployment

## Tech Stack

Frontend:

- React
- Vite
- TailwindCSS

Backend:

- FastAPI
- Gemini
- Qdrant
- Sentence Transformers
- SQLite
